*AI-Integrated Web-Based Data Analytics Dashboard*

Di project ini, kalian akan membuat website dashboard data yang tidak hanya menampilkan grafik, tapi juga punya *fitur chat dengan AI* .

*Aplikasi yang dibangun harus mampu:*
- Menampilkan visualisasi data dalam bentuk dashboard interaktif
- Mengolah dan menyajikan data dari dataset nyata (bersumber dari Kaggle)
- Menyediakan fitur AI Chat Assistant yang memungkinkan pengguna melakukan analisis data menggunakan bahasa natural

*Fitur* :
- KPI (Scorecard, summary)
- line chart
- bar chart
- pie chart
- table
- filter tanggal
- filter kategori

*Fitur Utama: AI-Based Data Interaction*
Mahasiswa wajib mengimplementasikan fitur chat berbasis AI yang dapat digunakan untuk:
- Menjelaskan isi visualisasi data
- Memberikan insight berdasarkan data
- Menjawab pertanyaan analitis dari pengguna

Contoh interaksi yang diharapkan:
- "Jelaskan tren pada line chart tersebut"
- "Apa insight utama dari data ini?"
- "Kapan nilai tertinggi dan terendah terjadi?"
AI harus memberikan jawaban berdasarkan data yang ditampilkan pada dashboard, bukan jawaban umum.

*Ketentuan Data:* 
- Dataset dari platform publik seperti Kaggle
- Dataset minimal terdiri dari 100 baris data
